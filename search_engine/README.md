# Search Engine

## Description
Console application for local search engine on text files. Indexes documents, processes queries, outputs ranked results.

## Technology Stack
- C++17
- CMake
- nlohmann/json
- Google Test

## How to Run
1. Build in CLion.
2. Run `search_engine`.
3. Check `answers.json` in build directory.

To run tests: Run `server_test` executable.
